Concept and syntax

Anatomy of HTML document

                      
<!doctype html>
<html lang="en-US">
  <head>
    <meta charset="utf-8" />
    <title>My test page</title>
  </head>
  <body>
    <p>This is my page</p>
  </body>
</html>
  • <!doctype html> declares HTML version 5 to browsers
  • <html> is the root element that surround the entire HTML document
  • <head> is where we can include link to style sheet...
  • <body> contains visible contents

datetime attribute tranlsate date and time into machine readable format. Below are valid datetime values

  • month and date: YYYY-MM, YYYY-MM-DD, MM-DD
  • week: YYYY-WWW (e.g. 2024-W12)
  • time: HH:MM, HH:MM:SS, HH.MM.SS.mmm
  • duration string: (e.g. P12DT7H12M13S) (e.g. 12d 7h 12mn 13s)

Structural Elements

<div>

a generic container used for group content together 

                  
<h1>Section 1</h1>
<p>This is a paragraph inside section 1</p>

<div style="background-color: lightblue; padding: 10px;">
  <h1>Section 2</h1>
  <p>This is a paragraph inside section 2</p>
</div>

Structural Elements

<head>

Represents a container for introductory content(site logo, title…) or navigation links of a webpage or section.

Structural Elements

<nav>

define a section of navigation like menu

                  
<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">About</a></li>
    <li><a href="#contact">Contact</a></li>
  </ul>
</nav>

Structural Elements

<section>

grouping a related content into a distinct section

Structural Elements

<article>

define a self-contained composition of a webpage that is usually understood without requiring context. e.g. blog post, product description, user comment.

Structural Elements

<aside>

represent a proportion of content that is indirectly related to main content such as sidebar…

Structural Elements

<footer>

define a footer of the document that usually content info like author, copyright or links to term of use

Structural Elements

<main>

define the main content in the body of the document excluding header, navigation link, footer and sidebars

Putting all Structural Elements together

                  
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Website Structure</title>
</head>
<body>
  <header>
    <h1>Welcome to My Website</h1>
    <p>Your go-to resource for web development tips!</p>
  </header>

  <nav>
    <ul>
      <li><a href="#home">Home</a></li>
      <li><a href="#services">Services</a></li>
      <li><a href="#contact">Contact</a></li>
    </ul>
  </nav>

  <main>
    <section id="services">
      <h2>Our Services</h2>
      <p>We provide top-notch web development and design services.</p>
    </section>

    <article>
      <h2>Latest Blog Post</h2>
      <p>Discover how HTML5 makes your websites more accessible.</p>
    </article>
  </main>

  <aside>
    <h3>Upcoming Events</h3>
    <p>Join our free webinar on modern web design trends.</p>
  </aside>

  <footer>
    <p>© 2024 My Website. All rights reserved.</p>
  </footer>
</body>
</html>
