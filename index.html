</html>
  • <!doctype html> declares HTML version 5 to browsers
  • <html> is the root element that surround the entire HTML document
  • <head> is where we can include link to style sheet...
  • <body> contains visible contents

datetime attribute tranlsate date and time into machine readable format. Below are valid datetime values

  • month and date: YYYY-MM, YYYY-MM-DD, MM-DD
  • week: YYYY-WWW (e.g. 2024-W12)

Putting all Structural Elements together

                  
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Website Structure</title>
</head>
<body>
  <header>
    <h1>Welcome to My Website</h1>
    <p>Your go-to resource for web development tips!</p>
  </header>

  <nav>
    <ul>
      <li><a href="#home">Home</a></li>
      <li><a href="#services">Services</a></li>
      <li><a href="#contact">Contact</a></li>
    </ul>
  </nav>

  <main>
    <section id="services">
      <h2>Our Services</h2>
      <p>We provide top-notch web development and design services.</p>
    </section>

    <article>
      <h2>Latest Blog Post</h2>
      <p>Discover how HTML5 makes your websites more accessible.</p>
    </article>
  </main>

  <aside>
    <h3>Upcoming Events</h3>
    <p>Join our free webinar on modern web design trends.</p>
  </aside>

  <footer>
    <p>© 2024 My Website. All rights reserved.</p>
  </footer>
</body>
</html>

  document.body.appendChild(template);
</script>
