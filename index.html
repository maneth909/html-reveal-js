Elements and Tags

  • Most element aresurrounded by opeing and closing tag

Concept and syntax

Nesting elements

Each element can be placed inside other elements 

                      
✅✅
<p>My cat is <strong>very</strong> cute.</p>
              
❌❌
<p>My cat is <strong>very cute.</p></strong>

Concept and syntax

Anatomy of HTML document

                      
<!doctype html>
<html lang="en-US">
  <head>
    <meta charset="utf-8" />
    <title>My test page</title>
  </head>
  <body>
    <p>This is my page</p>
  </body>
</html>
  • <!doctype html> declares HTML version 5 to browsers
  • <html> is the root element that surround the entire HTML document
  • <head> is where we can include link to style sheet...
  • <body> contains visible contents
